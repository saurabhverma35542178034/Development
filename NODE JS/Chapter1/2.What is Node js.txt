What is Node.js?
Node.js a C++ application (CLI) which can understand and run JavaScript code outside of the browser.
It provides a runtime environment built on Chrome's V8 JavaScript engine, enabling developers to
execute JavaScript on the server side.

What is Node.js Used For?
. Web Servers and APIs

. Command-Line Tools for example, (npm, TypeScript, Webpack, ESLint, Babel, Yarn, Create React
App, Vue CLI, Angular CLI, Prettier, and more )

. Internet of Things (loT)
